# 来自莉娜的礼物

## 题面描述

一个以 $0$ 到 $n$ 的数字构成的图案应该以 $n$ 为中心，数字越接近边缘越少。例如，当 $n=5$ 时，图案应如下图所示：
```
     0     
    010
   01210
  0123210
 012343210
01234543210
 012343210
  0123210
   01210
    010
     0
```
现在给定最大数字 $n$，按照要求输出图案。

数据范围为 $2\le n\le 9$ 。



## 输入格式

第一行包含一个整数 $n$ ( $2<=n<=9$)。

## 输出格式

输出给定 $n$ 的图片。 注意应该严格遵守每行第一个数字之前的空格数。 同一行中的每两个相邻数字应恰好用一个空格分隔。 每行末尾的最后一个数字后面不应有空格。

## 样例 #1

### 样例输入 #1

```
2
```

### 样例输出 #1

```
0
  0 1 0
0 1 2 1 0
  0 1 0
    0
```

## 样例 #2

### 样例输入 #2

```
3
```

### 样例输出 #2

```
0
    0 1 0
  0 1 2 1 0
0 1 2 3 2 1 0
  0 1 2 1 0
    0 1 0
      0
```
