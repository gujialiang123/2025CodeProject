如果前一个=这一个==/，就不输出

否则输出

然后最后一位判段

然而会WA在”/////“的点上

所以我们开一个变量看输出量

然后最后一位判段如果输出量==0也输出

```cpp
#include<iostream>
#include<cstring>
using namespace std;
int main(){
//上面应该很清楚此处不再说了
    string s;
    cin>>s;//输入
    int len=s.length(),n=0;
    for(int i=1;i<len;i++){
        if(!(s[i]==s[i-1]&&s[i]=='/')){++n;cout<<s[i-1];}//删/
    }
    if(s[len-1]!='/'||n==0)cout<<s[len-1];//特判
}
```
