原题：CF43c

难度：1300

算法：贪心

## 思路
根据3的整除特征，配对的两个数只要除以3的余数和等于0或3即可。

即两个3的倍数配对或除以3余2和除以3余1的两个数配对，配对后的数即是3的倍数。

用a[3]数组记录除以3的余数的三种情况的个数。

答案为：a[0]/2+min(a[1],a[2])


## 实现
```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
    int n;
    cin>>n;
    int a[3]={0},c;
    for(int i=1;i<=n;i++)
    {
    	cin>>c;
    	a[c%3]++;
    }
    cout<<a[0]/2+min(a[1],a[2]);
    return 0;
}
```
