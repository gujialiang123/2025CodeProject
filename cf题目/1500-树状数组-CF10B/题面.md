# 电影院

时间限制：$1s$ | 空间限制：$512MB$

## 题目描述
有一个电影院，大小为 $k^2$ ，然后有 $n$ 波人依次到来。

每一波人都想选择尽量靠中间的位置，就是这一波人都必须坐在同一行，并且有一个花费就是 $\vert x-zx \vert+\vert y-zy \vert$。

然后你需要给他们安排座位，使得花费最小，如果花费一样，就坐左前方。

求如何安排。

N不超过1000，K不超过99。

## 输入输出样例
### 样例输入 #1

```
2 1
1 1
```
### 样例输出 #1

```
1 1 1
-1
```

### 样例输入 #2

```
4 3
1 2 3 1
```
### 样例输出 #2

```
2 2 2
1 1 2
3 1 3
2 1 1
```
