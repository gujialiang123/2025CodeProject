# 幸运数字



## 【题目描述】
Petya喜欢数字，他将4和7视为幸运数字。如果一个数能由4与7组成，他就是幸运数字，只要能被幸运数字整除的也算幸运数字！现给定一个数字，判断它是否是幸运数。
## 【输入格式】
一个整数n(1<=n<=1000)
## 【输出格式】
若n是幸运数或接近幸运数，输出“YES”；否则输出“NO”。（“”不用输出）




## 样例 #1

### 样例输入 #1

```
47
```

### 样例输出 #1

```
YES
```

## 样例 #2

### 样例输入 #2

```
16
```

### 样例输出 #2

```
YES
```

## 样例 #3

### 样例输入 #3

```
78
```

### 样例输出 #3

```
NO
```
