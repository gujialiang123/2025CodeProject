# 猫咪派对（简易版）

## 题面

给定一个长度为n的序列，求从第一项起最长的序列，使得去掉其中任一元素后每个数出现的次数相同，求这个最长序列的长度x。



## 输入格式

第一行包含一个整数n，1<=n<=10^5。

第二行包含n个整数,表示这个序列，每个数在1-10之间。

## 输出格式

输出单个整数 x 。

## 样例 #1

### 样例输入 #1

```
13
1 1 1 2 2 2 3 3 3 4 4 4 5
```

### 样例输出 #1

```
13
```

## 样例 #2

### 样例输入 #2

```
5
10 2 5 4 1
```

### 样例输出 #2

```
5
```

## 样例 #3

### 样例输入 #3

```
1
10
```

### 样例输出 #3

```
1
```

## 样例 #4

### 样例输入 #4

```
7
3 2 1 1 4 5 1
```

### 样例输出 #4

```
6
```

## 样例 #5

### 样例输入 #5

```
6
1 1 1 2 2 2
```

### 样例输出 #5

```
5
```
