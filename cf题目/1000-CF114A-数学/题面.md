# 数值

## 题面描述

给出两个数 $k, l$。如果存在 $l = k ^ n$（$n$ 为正整数），则在第一行输出 `YES`，并在第二行输出 $n-1$，否则输出 `NO`。



## 输入格式

第一输入行包含整数 $k$ ，第二行包含整数 $l$ ( $2<=k,l<=2^{31}-1$ )。

## 输出格式

如果该数字属于集合 petriciumus cifera，则应在输出的第一行打印“YES”，否则打印“NO”。 如果该数字属于该集合，则在秒行上打印唯一的数字 - 数字 $ l $ 的重要性。1

## 样例 #1

### 样例输入 #1

```
5
25
```

### 样例输出 #1

```
YES
1
```

## 样例 #2

### 样例输入 #2

```
3
8
```

### 样例输出 #2

```
NO
```
