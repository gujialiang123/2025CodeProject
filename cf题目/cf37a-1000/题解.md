## 题解
原题：CF37a

难度：1000

算法：排序



## 分析

桶排序
用一棵数组来存k出现几次

寻找有多少个不同的数：
```cpp
if(!t[a[i]])ansb++;//若a[i]目前没有出现则更新答案
	t[a[i]]++;//将a[i]的桶更新
```
寻找出现次数最多的：
```cpp
ansa=max(ansa,t[a[i]]);//每次操作后更新答案即可
```
## 代码实现

```cpp
#include<bits/stdc++.h>
using namespace std;
int n,ansa,ansb,tot;
int a[1005];
map<int,int>t;
int main()
{
	scanf("%d",&n);
	for(int i=1;i<=n;i++){
		scanf("%d",&a[i]);
		if(!t[a[i]])ansb++;
		t[a[i]]++;
		ansa=max(ansa,t[a[i]]);	
	}
	printf("%d %d\n",ansa,ansb);
    return 0;
}
```
