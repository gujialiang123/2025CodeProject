# 位置的数量

## 题面描述

一个人站在一个总人数为n队列里，他前面有不少于a个人，后面有不多于b个人。求他有多少种可能的不同位置。 
## 输入格式
 输入n,a,b (0<=a,b,n<=100) 
## 输出格式 
一个数，即这个人有多少种可能的不同位置



## 样例 #1

### 样例输入 #1

```
3 1 1
```

### 样例输出 #1

```
2
```

## 样例 #2

### 样例输入 #2

```
5 2 3
```

### 样例输出 #2

```
3
```

## 提示

第一个样本中的可能位置是:2和3(如果我们从1开始编号)。



第二个样本是3 4 5。
