原题：CF124A

难度：入门，1000

算法：数学


简单分析可得：

1、这个人的前面不能少于a个人，那么他可以站在 a+1 至 n ，共计 n-a 个位置。

2、这个人的后面不能多于b个人，那么他可以站在 n-b 至 n ，共计 b+1 个位置。

所以我们直接输出两种情况中较小的那个即可。

```
#include <bits/stdc++.h>
using namespace std;

int main(){
	long long n,a,b;
	scanf("%lld%lld%lld",&n,&a,&b);
	printf("%lld",min(n-a,b+1));
	return 0;
}
```
