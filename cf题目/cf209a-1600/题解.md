原题：CF209a

难度：1600

算法：dp、数学

## 思路
考虑动态规划求解。仔细看题，因为题目中所提到的交替，所以 
dp[i]的状态只和dp[i-1]和dp[i-2]有关，退出状态转移方程
dp[i]=dp[i-1]+dp[i-2]+2


## 实现
```cpp
#include<cstdio>
const int MOD=1000000007;
const int MAXN=1e6+5;
int n,dp[MAXN];
void work(){
	for(int i=2;i<=n;i++){
		dp[i]=(dp[i-1]+dp[i-2]+2)%MOD;
	}
}
int main(){
	scanf("%d",&n);
	dp[1]=1;
	work();
	printf("%d",dp[n]);
}

```
