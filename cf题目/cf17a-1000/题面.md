# 诺尔德巴赫问题
## 题目描述

尼克对质数很感兴趣。有一次，他读到了哥德巴赫问题。它指出，每个大于 2 的偶数整数都可以表示为两个素数之和。这引起了尼克的注意，于是他决定自己发明一个问题，并称之为 诺德巴赫问题。由于尼克只对素数感兴趣，诺尔德巴赫问题指出，从 2 到 n 的素数中，至少有 k 个素数可以表示为三个整数之和：两个相邻的素数和 1 。例如，19 = 7 + 11 + 1 、或 13 = 5 + 7 + 1 。

如果两个质数之间没有其他质数，则称这两个质数为邻质数。

你要帮助尼克，找出他是对还是错。

## 输入格式

输入的第一行包含两个整数 n ( 2 ≤ n ≤ 1000 ) 和 k ( 0 ≤ k ≤ 1000 )。

## 输出格式

如果从 2 到 n 的至少 k 个质数(包括 2 和 n )可以按上述方法表示，则输出YES。否则输出 NO 。

## 样例 #1

### 样例输入 #1

```
27 2
```

### 样例输出 #1

```
YES
```

## 样例 #2

### 样例输入 #2

```
45 7
```

### 样例输出 #2

```
NO
```

## 提示

在第一个示例中，答案是YES，因为至少有两个数可以按描述的形式表示(例如 13 和 19)。在第二个示例中，答案是 NO，因为从 2 到 45 的 7 个质数不可能用所需的形式表示。
