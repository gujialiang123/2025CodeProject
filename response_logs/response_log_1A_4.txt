## Complete Solution: Calculating the Minimum Number of Flagstones Needed to Pave a Theatre Square
### Problem Overview
Given a rectangular square with dimensions `n` by `m` meters, we need to find the minimum number of square granite flagstones, each with dimensions `a` by `a` meters, required to pave the entire square.

### Solution Approach
To solve this problem, we will:
1. Import necessary modules and define variables for the input values.
2. Calculate the minimum number of flagstones needed to cover the length and width of the square.
3. Compute the total number of flagstones required to pave the entire square.

### Code Solution

```python
import math

def calculate_flagstones(n, m, a):
    """
    Calculate the minimum number of flagstones needed to pave a rectangular square.

    Args:
    n (int): Length of the square.
    m (int): Width of the square.
    a (int): Side length of the flagstones.

    Returns:
    int: Minimum number of flagstones required.
    """
    # Calculate the minimum number of flagstones needed to cover the length and width
    flagstones_length = math.ceil(n / a)
    flagstones_width = math.ceil(m / a)

    # Compute the total number of flagstones required
    total_flagstones = flagstones_length * flagstones_width

    return total_flagstones

def main():
    # Input values
    n, m, a = map(int, input().split())

    # Calculate and print the minimum number of flagstones
    min_flagstones = calculate_flagstones(n, m, a)
    print(min_flagstones)

if __name__ == "__main__":
    main()
```

### Explanation
*   The `calculate_flagstones` function calculates the minimum number of flagstones needed to cover the length and width of the square using the `math.ceil` function, which returns the smallest integer not less than the given number.
*   The `main` function reads the input values, calls the `calculate_flagstones` function, and prints the result.

### Example Use Case
Input:
```
6 9 4
```
Output:
```
6
```
In this example, the square has a length of 6 meters and a width of 9 meters, and the flagstones have a side length of 4 meters. The minimum number of flagstones required to pave the entire square is 6.
