### Problem Solution: Tanya's Subtraction Algorithm

Here is a complete solution to the problem using Python:

```python
def tanya_subtraction(n, k):
    """
    Simulates Tanya's subtraction algorithm.

    Args:
    n (int): The initial number.
    k (int): The number of subtractions.

    Returns:
    int: The result after all subtractions.
    """
    for _ in range(k):
        # Check if the last digit of the number is non-zero
        if n % 10 != 0:
            n -= 1  # Decrease the number by one
        else:
            n //= 10  # Remove the last digit
    return n

# Read input
n, k = map(int, input().split())

# Calculate and print the result
result = tanya_subtraction(n, k)
print(result)
```

### Explanation

1. Define a function `tanya_subtraction` that takes two arguments: `n` (the initial number) and `k` (the number of subtractions).
2. Inside the function, use a `for` loop to repeat the subtraction process `k` times.
3. In each iteration, check if the last digit of the number is non-zero by checking `n % 10 != 0`.
   * If it is non-zero, decrease the number by one using `n -= 1`.
   * If it is zero, remove the last digit using integer division `n //= 10`.
4. After all subtractions, return the final result.
5. Read the input values `n` and `k` from the user.
6. Call the `tanya_subtraction` function with the input values and print the result.
